# 🔐 تولیدکنندهٔ رمز عبور امن | Password Generator

<div dir="rtl">

تولیدکنندهٔ رمز عبور امن و سریع با قابلیت شخصی‌سازی کامل.  
امنیت دیجیتال شما از این به بعد دیگر بهانه‌ای برای «پسورد ضعیف» ندارد.  
(هر چند اگر هنوز هم از `123456` استفاده کنی، خدا به همرات…)

</div>

---

## 🚀 ویژگی‌ها | Features

- **تولید رمز کاملاً تصادفی** با آنتروپی بالا (خداحافظ حدس‌زدنی‌ها)
- **انتخاب طول دلخواه** (از ۸ تا ۱۲۸ کاراکتر، بستگی داره چقدر پارانویا داری)
- **حروف بزرگ و کوچک، اعداد و نمادها** – همه چی رو می‌تونی تنظیم کنی
- **نمایش قدرت رمز** (ضعیف / متوسط / قوی / Fort Knox)
- **خروجی به کلیپ‌بورد** با یک کلیک، چون تایپ کردن ۶۴ کاراکتر رندوم شکنجه‌ست
- **واسط کاربری مینیمال و زیبا** (اگر عاشق CLI هستی، نسخهٔ ترمینال هم موجوده)
- **آپشن batch generation** برای مواقعی که می‌خوای هزارتا رمز یکجا بسازی و بعد قاطی کنی
- **بدون اتصال به اینترنت** – چون پاره شدن از اینترنت هم امنیت محسوب می‌شه
- **کاملاً متن‌باز** – کد رو ببین، مطمئن شو، بعد اعتماد کن

## 📦 نصب | Installation

برای نصب پیش‌نیازها و اجرای برنامه:
```bash
git clone https://github.com/atnsad/password-generator.git
cd password-generator
pip install -r requirements.txt
python main.py

> اگر از ویندوز استفاده می‌کنی و هنوز Python نداری… راستش رو بخوای، این برنامه کوچک‌ترین مشکلت نیست.

## 🛠️ نحوهٔ استفاده | Usage

### واسط گرافیکی (GUI)

فقط اجراش کن، اسلایدرها رو بکش، دکمه‌ها رو بزن، رمزت آماده‌ست.  
ساده‌تر از باز کردن در یخچال نیمه‌شب.

### خط فرمان (CLI)

bash
python passgen.py --length 24 --no-symbols

| آرگومان           | توضیح                                      |
|-------------------|--------------------------------------------|
| `--length`        | طول رمز (پیش‌فرض: ۲۰)                      |
| `--no-uppercase`  | حروف بزرگ نیاد – چرا؟ فقط تو می‌دونی        |
| `--no-digits`     | عدد نیاد                                   |
| `--no-symbols`    | نماد نیاد (برای وب‌سایت‌های عقب‌افتاده)     |
| `--batch 10`      | ده تا رمز بریزه بیرون پشت سر هم             |

## 🧠 امنیت | Security

از ماژول `secrets` پایتون استفاده می‌شه، نه `random`.  
یعنی «تصادفی واقعاً تصادفی» – اگه NSA هم بیاد، شاید موفق بشه، ولی خب کی NSA رو دعوت کرده؟

- هیچ رمزی ذخیره نمی‌شه (حتی تو مموری بیشتر از چند ثانیه)
- هیچ اطلاعاتی به بیرون درز نمی‌کنه، چون چیز خاصی برای درز کردن نداریم
- از CSPRNG سیستم‌عامل مستقیماً تغذیه می‌شه
