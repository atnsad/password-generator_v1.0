# 🔐 Password Generator (تولیدکننده رمز عبور پیشرفته)

خوش آمدید! این پروژه، حاصلِ ترکیبِ دقت در برنامه‌نویسی و تمرکز بر امنیت سایبری است. این ابزار نه فقط یک تولیدکننده رمز عبور معمولی، بلکه یک راهکار مهندسی‌شده برای مقابله با حملات Brute Force و دیکشنری است که با هدفِ ایجاد بالاترین سطحِ امنیت برای حساب‌های کاربری شما طراحی شده است. 🛡️

---

### 💡 فلسفه وجودی این ابزار
در دنیایِ پرخطرِ امروز، استفاده از رمزهای عبور تکراری یا ساده، حکمِ باز گذاشتنِ درِ خانه را دارد. این برنامه با استفاده از الگوریتم‌های استانداردِ رمزنگاری و تولید اعداد تصادفی (Cryptographically Secure Pseudo-Random Number Generation - CSPRNG)، پسوردهایی تولید می‌کند که از نظر تئوری، شکستنِ آن‌ها توسط ابرکامپیوترهای فعلی، قرن‌ها زمان می‌برد. ⏳

ما باور داریم که "امنیت" نباید پیچیده باشد؛ بنابراین این ابزار با رابط کاربری مینیمال و منطقِ backend قدرتمند، این پیچیدگی را به سادگیِ یک کلیک تبدیل کرده است. 🚀

---

### ✨ قابلیت‌ها و ویژگی‌های کلیدی
این ابزارِ جمع‌وجور اما توانمند، امکانات زیر را در اختیار شما قرار می‌دهد:

*   **آنتروپی بالا (High Entropy):** ترکیبِ کاملاً تصادفی از حروف کوچک، حروف بزرگ، اعداد و نمادهای خاص (Special Characters) که قدرتِ رمز عبور را به صورت نمایی افزایش می‌دهد. 🎲
*   **کنترلِ کامل (Granular Control):** شما تصمیم می‌گیرید! امکان تعیین دقیقِ طولِ پسورد (مثلاً ۳۲ کاراکتر یا بیشتر) و تعیینِ کاراکترهای مجاز، این ابزار را برای هر نوع سیستم امنیتی ایده‌آل می‌کند. ⚙️
*   **حریم خصوصی مطلق (Zero-Knowledge):** تمامیِ عملیاتِ تولیدِ پسورد به صورتِ کاملاً محلی (Local) روی سیستم شما انجام می‌شود. هیچ داده‌ای به سرورهای خارجی ارسال نمی‌شود و هیچ ردی از رمزهای عبور شما در اینترنت باقی نمی‌ماند. امنیتِ داده‌های شما، خط قرمزِ این پروژه است. 🔒
*   **سرعتِ بی‌نظیر:** استفاده از زبان‌های سطح‌بالا و بهینه‌سازیِ کدهایِ مولد، باعث شده تا تولیدِ صدها پسورد در کسری از ثانیه انجام شود. ⚡
